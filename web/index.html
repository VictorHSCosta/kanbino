Instalação Simples

Comece a usar o Gemkanbino em poucos segundos

1

Instale a Gem

gem install gemkanbino
2

Verifique a Instalação

gemkanbino --version
3

Explore os Comandos

gemkanbino --help

Requisitos

  • Ruby 2.7 ou superior
  • Sistema operacional: macOS, Linux ou Windows (WSL)
  • Terminal de linha de comando

Comandos Disponíveis

Explore todos os comandos e opções disponíveis

version

Exibe a versão atual do Gemkanbino

gemkanbino version
--version -v

pwd

Mostra o diretório de trabalho atual

gemkanbino pwd

ls

Lista arquivos e diretórios no caminho especificado

gemkanbino ls [PATH]
--all, -a Mostrar arquivos ocultos
--long, -l Usar formato de lista longa

cd

Muda o diretório atual para o caminho especificado

gemkanbino cd PATH

select

Seleciona um arquivo para operações subsequentes

gemkanbino select FILE

info

Exibe informações detalhadas sobre o arquivo selecionado ou específico

gemkanbino info [FILE]

copy

Copia o arquivo selecionado ou específico para armazenamento local

gemkanbino copy [FILE]
--target, -t Nome do diretório alvo

upload

Envia o arquivo selecionado ou específico para a nuvem

gemkanbino upload [FILE]
--provider, -p Provedor de upload (fileio, transfersh)

list

Lista todos os arquivos armazenados

gemkanbino list

config

Exibe ou define configurações da ferramenta

gemkanbino config [KEY] [VALUE]

interactive

Inicia o modo shell interativo

gemkanbino interactive

Exemplos de Uso

Veja como usar o Gemkanbino em situações do dia a dia

Navegando e Listando Arquivos

# Listar arquivos no diretório atual
gemkanbino ls

# Listar com detalhes
gemkanbino ls -l

# Mudar para diretório Documents
gemkanbino cd Documents

# Ver onde estamos
gemkanbino pwd

Gerenciando Arquivos

# Selecionar um arquivo
gemkanbino select document.pdf

# Ver informações do arquivo
gemkanbino info

# Copiar para armazenamento local
gemkanbino copy -t backups

# Upload para a nuvem
gemkanbino upload -p fileio

Configuração e Uso Avançado

# Ver todas as configurações
gemkanbino config

# Definir provedor de upload padrão
gemkanbino config upload_provider fileio

# Listar arquivos armazenados
gemkanbino list

# Iniciar modo interativo
gemkanbino interactive
